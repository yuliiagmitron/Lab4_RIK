# SurveyApp — Опитування "Задоволеність сервісом"

Мобільний додаток-опитування, створений на **React Native** з використанням **Expo**.
Лабораторна робота №4.

## Опис

Додаток містить форму опитування про задоволеність сервісом із 7 запитань різних типів:

| #   | Запитання                          | Тип                        |
| --- | ---------------------------------- | -------------------------- |
| 1   | Оцінка загальної якості сервісу    | Текстове поле              |
| 2   | Швидкість відповіді на запит       | Текстове поле              |
| 3   | Які аспекти сервісу сподобались    | Чекбокси (множинний вибір) |
| 4   | Як дізнались про сервіс            | Чекбокси (множинний вибір) |
| 5   | Які послуги використовували        | Мультивибір                |
| 6   | Що потребує покращення             | Мультивибір                |
| 7   | Чи порекомендували б сервіс друзям | Текстове поле              |

Після заповнення та натискання кнопки **"Надіслати"** відповіді зберігаються у файл `survey_results.txt`, який автоматично завантажується на диск.

## Технології

- **React Native** — фреймворк для мобільної розробки
- **Expo** — платформа для швидкого запуску та тестування
- **Expo Router** — файлова маршрутизація
- **JavaScript**

## Структура проєкту

```
SurveyApp/
```

## Запуск

### Передумови

- Node.js (v18+)
- npm

### Встановлення та запуск

```bash
# Перейти у папку проєкту
cd SurveyApp

# Встановити залежності (якщо ще не встановлені)
npm install

# Запустити у браузері
npx expo start --web

# Запустити через Expo Tunnel
npx expo start --tunnel
```

Додаток відкриється у браузері за адресою `http://localhost:8081`.

## Збереження результатів

Результати зберігаються у текстовому форматі `.txt`.
